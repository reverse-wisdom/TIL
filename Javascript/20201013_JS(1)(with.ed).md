### 식별자(indentifier)

- 변수명은 식별자라고도 불림

- 규칙

  1. 반드시 **문자**, **달라($)** 또는 밑줄(_)로 시작해야한다(숫자나, `-` 로 시작할 수 없다)

     ```
     const hi-hello (중간에 들어가도안됨)
     ```

     

  2. 대소문자를 구분한다.
  3. 예약어는 사용할 수 없다(const, function ,class, ...)

- 스타일

  - 카멜케이스(lowerCamelCase)
  - - 객체, 변수, 함수

  - 파스칼 케이스(UpperCamelCase)
    - 클래스 생성자
  - 대문자 스네이크 케이스(UPPER_CASE)
    - 상수: 변수나 변수의 속성이 변하지 않는 것
